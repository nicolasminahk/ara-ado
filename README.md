# YESAN / ARAÑADO landing v2

Prototipo HTML responsive generado desde el PDF de maqueta.

## Uso
Abrir `index.html` en navegador o servir la carpeta con cualquier servidor estático.

## Assets
Las imágenes base fueron extraídas del PDF y están en `assets/bg/`.

## Tipografías
El PDF usa familias HelveticaNowDisplay-Bd y Helvetica LT Pro. Por licencia, no se extraen ni se redistribuyen fuentes embebidas desde el PDF. Para igualar 1:1, colocar fuentes licenciadas en `assets/fonts/` y agregar los `@font-face` correspondientes arriba del CSS.
